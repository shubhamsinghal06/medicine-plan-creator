## How to Use

1. Open `index.html` in your web browser
2. Enter medicines for each meal time:
   - Type the medicine name
   - Add dosage information (e.g., "1 tablet", "2 capsules")
   - Click "+ Add Medicine" to add more medicines for that meal
3. Optionally fill in:
   - Patient name
   - Date (defaults to today)
   - Additional notes or instructions
4. Click "ğŸ“„ Generate & Download PDF" to create and download your medicine plan
